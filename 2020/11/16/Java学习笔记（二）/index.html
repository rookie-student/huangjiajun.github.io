Java学习笔记（二）

发表于 2020-11-16

前言

Java基本语法 — 运算符和表达式

正文

运算符总共有六类

  1. 算数运算符
  2. 逻辑运算符
  3. 赋值运算符
  4. 关系运算符
  5. 位运算符
  6. 条件运算符（三目运算符）

算数运算符

+ - * / % ++ –

/：结果取整

%：取余

++：自加1

​ a++表示先使用a的值，再自加1

​ ++a表示先自加1，再使用a的值

–类似于++

逻辑运算符(运算结果为布尔型)

& | ! && ||

&：逻辑与，两边同时为真结果为真，一边为假肯定为假

|：逻辑或，两边同时为假结果为假，一边为真肯定为真

！：非，取反

&&：短路与，只要有一边为假，JVM就不会判断另一边的值，结果直接为假，效率高于&

||：短路或，只要有一边为真，JVM就不会判断另一边的值，结果直接为真，效率高于|

关系运算符(运算结果为布尔型)

> < == >= <= !=

赋值运算符

= += -= *= /=

位运算符

~ & | ^ >> << <<<

：按位取反，1100=0011，1100为二进制，如果是其他进制数字，先转换成二进制再进行计算

&：按位相与，101 & 110=100

|：按位或，101 | 110 =111

^：异或，都为1或者都为0取0，不相同时取1，101 ^ 110=011

>>：保留符号位右移，>>n右移n位，实际是右移(n%32)位，若为长整型，右移（n%64）位，左边空出来的位用符号位代替，效率高于通常的除法运算

<<：左移，、<<n左移n位，实际是左移（n%32）位，若为长整型，左移（n%64）位，右边空出来的位用0代替，效率高于乘法运算

>>>：不保留符号右移，左边空出来的位用0代替，其他的与>>类似

条件运算符（三目运算符）

?:

e1?e2:e3

e1是一个布尔表达式，如果e1的值为true，则计算e2的值，否则计算e3的值，e2和e3需要返回相同的类型或兼容的数据类型，且该类型不能是void
